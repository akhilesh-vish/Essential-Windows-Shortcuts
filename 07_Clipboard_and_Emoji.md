# 📋 Clipboard & Emoji

The Windows Clipboard and Emoji Panel are powerful built-in tools designed to improve the way you copy, paste, and communicate. Clipboard History lets you access multiple recently copied items instead of just the last one, while the Emoji Panel provides quick access to emojis, GIFs, kaomoji, symbols, and special characters.

Whether you're writing documents, chatting with colleagues, coding, or creating content, these shortcuts can help you work faster and express yourself more effectively.

> [!TIP]
> Enable **Clipboard History** before using it. Once activated, you'll never have to worry about losing previously copied text or images again.

# ⌨️ Keyboard Shortcuts

| Shortcut                      | Description                                                          | Availability    |
| ----------------------------- | -------------------------------------------------------------------- | --------------- |
| **Ctrl + C**                  | Copy the selected text, file, image, or other item to the clipboard. | Windows 10 / 11 |
| **Ctrl + X**                  | Cut the selected item and place it on the clipboard.                 | Windows 10 / 11 |
| **Ctrl + V**                  | Paste the most recently copied or cut item.                          | Windows 10 / 11 |
| **Win + V**                   | Open Clipboard History to view and paste recently copied items.      | Windows 10 / 11 |
| **Win + . (Period)**          | Open the Emoji Panel with emojis, GIFs, kaomoji, and symbols.        | Windows 10 / 11 |
| **Win + ; (Semicolon)**       | Alternative shortcut for opening the Emoji Panel.                    | Windows 10 / 11 |
| **Win + H**                   | Start Voice Typing (Dictation) for hands-free text entry.            | Windows 10 / 11 |
| **Ctrl + A**                  | Select all content in the current document or text field.            | Windows 10 / 11 |
| **Ctrl + Z**                  | Undo the previous action.                                            | Windows 10 / 11 |
| **Ctrl + Y**                  | Redo the previously undone action.                                   | Windows 10 / 11 |
| **Ctrl + Backspace**          | Delete the previous word instead of a single character.              | Windows 10 / 11 |
| **Ctrl + Delete**             | Delete the next word.                                                | Windows 10 / 11 |
| **Ctrl + Left Arrow**         | Move the cursor to the beginning of the previous word.               | Windows 10 / 11 |
| **Ctrl + Right Arrow**        | Move the cursor to the beginning of the next word.                   | Windows 10 / 11 |
| **Shift + Arrow Keys**        | Select text one character or line at a time.                         | Windows 10 / 11 |
| **Ctrl + Shift + Arrow Keys** | Select an entire word at a time.                                     | Windows 10 / 11 |
| **Home**                      | Move the cursor to the beginning of the current line.                | Windows 10 / 11 |
| **End**                       | Move the cursor to the end of the current line.                      | Windows 10 / 11 |
| **Ctrl + Home**               | Move to the beginning of the document.                               | Windows 10 / 11 |
| **Ctrl + End**                | Move to the end of the document.                                     | Windows 10 / 11 |

# ⭐ Most Useful Shortcuts

These are the shortcuts you'll use most often when working with text, documents, emails, or messaging applications.

| Shortcut                      | Why It's Useful                                                        |
| ----------------------------- | ---------------------------------------------------------------------- |
| **Ctrl + C / Ctrl + V**       | Quickly copy and paste text, files, and images.                        |
| **Win + V**                   | Access Clipboard History instead of relying on a single copied item.   |
| **Win + .**                   | Insert emojis, symbols, GIFs, and kaomoji into almost any application. |
| **Win + H**                   | Dictate text instead of typing manually.                               |
| **Ctrl + Backspace**          | Delete entire words quickly while editing text.                        |
| **Ctrl + Arrow Keys**         | Move through documents one word at a time for faster navigation.       |
| **Ctrl + Shift + Arrow Keys** | Select entire words for quicker editing.                               |
| **Ctrl + Home / End**         | Jump instantly to the beginning or end of long documents.              |

# 💡 Productivity Tips

* Enable **Clipboard History** and use **Win + V** whenever you're copying multiple pieces of information between applications.
* Pin frequently used clipboard items so they're always available, even after copying something else.
* Use the **Emoji Panel** not only for emojis but also for **special symbols**, **mathematical operators**, **currency symbols**, and **kaomoji**.
* Voice Typing (**Win + H**) can significantly speed up writing emails, notes, and documentation.
* Learn **Ctrl + Arrow Keys** and **Ctrl + Backspace** to edit text much faster without reaching for the mouse.
* Combine Clipboard History with cloud synchronization (if enabled) to access copied content across multiple Windows devices.

# 📝 Notes

> [!NOTE]
> Clipboard History is disabled by default on a new Windows installation and must be enabled the first time you press **Win + V**.

> [!NOTE]
> The Emoji Panel works in most modern applications, including web browsers, Microsoft Office, messaging apps, and many third-party programs.

> [!NOTE]
> Voice Typing accuracy depends on your microphone quality, background noise, and the selected language.

> [!NOTE]
> Clipboard History stores multiple recently copied items, but very large files and some specialized data formats may not appear in the history.

# 💭 Did You Know?

> 💡 **Clipboard History can store more than just text.**
> It can also keep copied images and frequently used snippets, making it much easier to reuse content throughout your day.

---

> 💡 **You can pin clipboard items.**
> Frequently copied information such as email addresses, code snippets, phone numbers, or signatures can be pinned so they remain available even after restarting your computer.

---

> 💡 **The Emoji Panel is more than just emojis.**
> In addition to emojis, it provides access to **GIFs**, **kaomoji**, **Unicode symbols**, **currency signs**, **punctuation**, and many other special characters.

---

> 💡 **Voice Typing automatically adds punctuation.**
> When enabled in Voice Typing settings, Windows can insert commas, periods, and question marks automatically while you speak.
