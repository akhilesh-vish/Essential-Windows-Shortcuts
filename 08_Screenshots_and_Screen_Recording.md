# 📸 Screenshots & Screen Recording

Capturing screenshots and recording your screen are essential skills for troubleshooting, creating tutorials, sharing information, documenting issues, or saving important content. Windows includes several built-in tools that let you capture your entire screen, a specific window, or a selected region—all without installing additional software.

This chapter covers the most useful keyboard shortcuts for taking screenshots, using the Snipping Tool, and recording your screen with Xbox Game Bar.

> [!TIP]
> Learn the difference between copying a screenshot to the clipboard and automatically saving it as a file. Knowing which shortcut to use can save you time and prevent unnecessary steps.

# ⌨️ Keyboard Shortcuts

| Shortcut               | Description                                                                                                                     | Availability    |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------- | --------------- |
| **PrtScn**             | Capture the entire screen and copy it to the clipboard. Depending on your Windows settings, it may also open the Snipping Tool. | Windows 10 / 11 |
| **Alt + PrtScn**       | Capture only the currently active window and copy it to the clipboard.                                                          | Windows 10 / 11 |
| **Win + PrtScn**       | Capture the entire screen and automatically save it to the **Pictures > Screenshots** folder.                                   | Windows 10 / 11 |
| **Win + Shift + S**    | Open the Snipping Tool overlay to capture a rectangular, freeform, window, or full-screen snip.                                 | Windows 10 / 11 |
| **Win + G**            | Open Xbox Game Bar to access screen recording, screenshots, audio controls, and performance monitoring.                         | Windows 10 / 11 |
| **Win + Alt + R**      | Start or stop screen recording using Xbox Game Bar.                                                                             | Windows 10 / 11 |
| **Win + Alt + PrtScn** | Capture a screenshot of the active application using Xbox Game Bar.                                                             | Windows 10 / 11 |
| **Win + Alt + G**      | Record the previous 30 seconds of gameplay or supported application activity (Background Recording must be enabled).            | Windows 10 / 11 |
| **Win + Alt + M**      | Turn the microphone on or off while recording with Xbox Game Bar.                                                               | Windows 10 / 11 |
| **Win + Alt + B**      | Enable or disable HDR while gaming (supported hardware only).                                                                   | Windows 11      |
| **Esc**                | Close the Snipping Tool overlay or Xbox Game Bar interface without capturing.                                                   | Windows 10 / 11 |

# 💡 Productivity Tips

* Use **Win + Shift + S** when you only need part of your screen. It reduces the need for cropping later.
* If you frequently create tutorials or documentation, **Win + PrtScn** automatically saves screenshots, making them easier to organize.
* Use **Alt + PrtScn** instead of cropping a full-screen capture when you only need the active application window.
* Screen recordings made with **Xbox Game Bar** are automatically saved to the **Videos > Captures** folder.
* Turn on your microphone with **Win + Alt + M** when recording presentations or tutorials that require narration.
* If you often troubleshoot technical issues, combine screenshots with the Snipping Tool's annotation features to highlight important details.

# 📝 Notes

> [!NOTE]
> The **Snipping Tool** replaces the legacy Snip & Sketch application in modern versions of Windows.

> [!NOTE]
> Xbox Game Bar is primarily designed to record applications and games. It cannot record certain Windows desktop elements, such as File Explorer or the desktop itself, in every scenario.

> [!NOTE]
> The behavior of the **PrtScn** key can be customized in **Settings → Accessibility → Keyboard**, where it can be configured to open the Snipping Tool.

> [!NOTE]
> Background recording (**Win + Alt + G**) must be enabled in Xbox Game Bar settings before it can capture previous gameplay.

# ⚙️ Requirements

| Requirement              | Details                                                                  |
| ------------------------ | ------------------------------------------------------------------------ |
| **Operating System**     | Windows 10 or Windows 11                                                 |
| **Snipping Tool**        | Included with Windows and required for **Win + Shift + S**.              |
| **Xbox Game Bar**        | Must be installed and enabled for recording shortcuts.                   |
| **Storage Space**        | Required for saving screenshots and video recordings.                    |
| **Background Recording** | Must be enabled to use **Win + Alt + G**.                                |
| **HDR Support**          | Compatible display and graphics hardware required for **Win + Alt + B**. |

# 💭 Did You Know?

> 💡 **Your screenshots are automatically organized.**
> Screenshots captured with **Win + PrtScn** are automatically saved in the **Pictures > Screenshots** folder, making them easy to find later.

---

> 💡 **The Snipping Tool offers multiple capture modes.**
> Press **Win + Shift + S** to choose between rectangular, freeform, window, or full-screen captures depending on your needs.

---

> 💡 **Xbox Game Bar isn't just for gamers.**
> Many professionals use it to record software demonstrations, presentations, tutorials, and application workflows without installing third-party recording software.

---

> 💡 **You can copy or save—it's your choice.**
> **PrtScn** copies your screenshot to the clipboard, while **Win + PrtScn** saves it directly as an image file. Choosing the right shortcut can streamline your workflow.
