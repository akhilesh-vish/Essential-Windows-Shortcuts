# 🪟 Desktop & Window Management Shortcuts

Efficiently managing open windows is one of the easiest ways to improve your productivity in Windows. Instead of constantly reaching for your mouse, these keyboard shortcuts let you switch between applications, organize your workspace, snap windows into place, and manage multiple virtual desktops with ease.

Whether you're working on a single monitor or a multi-monitor setup, mastering these shortcuts will help you stay focused and multitask more efficiently.

> [!TIP]
> Combine these shortcuts with **Virtual Desktops** to create dedicated workspaces for work, study, gaming, or entertainment.

# ⌨️ Keyboard Shortcuts

| Shortcut                   | Description                                                                                                           | Availability    |
| -------------------------- | --------------------------------------------------------------------------------------------------------------------- | --------------- |
| **Alt + Tab**              | Switch between all open applications. Hold **Alt** and repeatedly press **Tab** to cycle through them.                | Windows 10 / 11 |
| **Alt + Shift + Tab**      | Cycle through open applications in reverse order.                                                                     | Windows 10 / 11 |
| **Win + Tab**              | Open Task View to see all open windows and virtual desktops.                                                          | Windows 10 / 11 |
| **Win + D**                | Minimize all open windows and show the desktop. Press again to restore them.                                          | Windows 10 / 11 |
| **Win + M**                | Minimize every open window.                                                                                           | Windows 10 / 11 |
| **Win + Shift + M**        | Restore all windows minimized using **Win + M**.                                                                      | Windows 10 / 11 |
| **Win + Home**             | Minimize all windows except the currently active one. Press again to restore them.                                    | Windows 10 / 11 |
| **Win + ↑**                | Maximize the active window.                                                                                           | Windows 10 / 11 |
| **Win + ↓**                | Restore a maximized window or minimize it if it's already restored.                                                   | Windows 10 / 11 |
| **Win + ←**                | Snap the active window to the left half of the screen.                                                                | Windows 10 / 11 |
| **Win + →**                | Snap the active window to the right half of the screen.                                                               | Windows 10 / 11 |
| **Win + Shift + ←**        | Move the active window to the monitor on the left.                                                                    | Windows 10 / 11 |
| **Win + Shift + →**        | Move the active window to the monitor on the right.                                                                   | Windows 10 / 11 |
| **Win + Shift + ↑**        | Stretch the active window vertically while keeping its current width.                                                 | Windows 10 / 11 |
| **Win + Ctrl + D**         | Create and switch to a new virtual desktop.                                                                           | Windows 10 / 11 |
| **Win + Ctrl + ←**         | Switch to the virtual desktop on the left.                                                                            | Windows 10 / 11 |
| **Win + Ctrl + →**         | Switch to the virtual desktop on the right.                                                                           | Windows 10 / 11 |
| **Win + Ctrl + F4**        | Close the current virtual desktop. Open windows move to the previous desktop.                                         | Windows 10 / 11 |
| **Win + , (Comma)**        | Temporarily peek at the desktop while the keys are held down.                                                         | Windows 10 / 11 |
| **Win + Ctrl + Shift + B** | Restart the graphics driver. Useful if the display freezes or goes black temporarily.                                 | Windows 10 / 11 |
| **Alt + F4**               | Close the currently active window or application. If no applications are open, displays the Shut Down Windows dialog. | Windows 10 / 11 |
| **F11**                    | Toggle Full Screen mode in supported applications such as File Explorer and most web browsers.                        | Windows 10 / 11 |

# ⭐ Most Useful Shortcuts

These are the shortcuts you'll probably use every day while working on your computer.

| Shortcut                | Why It's Useful                                                     |
| ----------------------- | ------------------------------------------------------------------- |
| **Alt + Tab**           | Quickly switch between open applications without using the taskbar. |
| **Win + ← / →**         | Instantly snap windows side by side for multitasking.               |
| **Win + ↑**             | Maximize the current window.                                        |
| **Win + D**             | Show the desktop without manually minimizing each window.           |
| **Win + Home**          | Focus on your current task by minimizing every other window.        |
| **Win + Ctrl + D**      | Create a separate workspace for another task or project.            |
| **Win + Ctrl + ← / →**  | Move between virtual desktops almost instantly.                     |
| **Alt + F4**            | Close the current application quickly.                              |
| **Win + Shift + ← / →** | Move windows between monitors without dragging them.                |
| **Win + Tab**           | View all open windows and desktops in one place.                    |

# 💡 Productivity Tips

* Use **Snap** (`Win + ←` and `Win + →`) to compare documents, spreadsheets, or webpages side by side.
* Create separate **Virtual Desktops** for work, personal tasks, and entertainment to reduce distractions.
* If you're using multiple monitors, **Win + Shift + ← / →** is much faster than dragging windows across screens.
* Instead of minimizing every window individually, press **Win + D** to instantly access your desktop.
* Working on one important task? Press **Win + Home** to minimize everything except the active window.
* If your screen suddenly turns black or freezes after a graphics issue, try **Win + Ctrl + Shift + B** before restarting your PC.

# 📝 Notes

> [!NOTE]
> Snap shortcuts work best when **Snap Windows** is enabled in Windows Settings.

> [!NOTE]
> Some window management features are enhanced in Windows 11, including **Snap Layouts**, which are covered in a later chapter.

> [!NOTE]
> Virtual Desktops allow each desktop to have its own collection of open applications, helping you organize different workflows.

> [!NOTE]
> Full Screen mode (**F11**) is supported by many applications, but not all software uses this shortcut.

# ⚙️ Requirements

| Requirement           | Details                                                                                      |
| --------------------- | -------------------------------------------------------------------------------------------- |
| **Operating System**  | Windows 10 or Windows 11                                                                     |
| **Snap Windows**      | Enable **Settings → System → Multitasking → Snap windows** for the best snapping experience. |
| **Multiple Monitors** | Required for monitor movement shortcuts such as **Win + Shift + ← / →**.                     |
| **Virtual Desktops**  | Built into Windows 10 and Windows 11; no additional software required.                       |
| **Graphics Driver**   | **Win + Ctrl + Shift + B** only restarts the graphics driver if Windows is responsive.       |

# 💭 Did You Know?

> 💡 **You can have multiple desktops for different tasks.**
> Create one desktop for work, another for personal browsing, and a third for entertainment or gaming. Switching between them takes just a second with **Win + Ctrl + ← / →**.

---

> 💡 **Snap windows without dragging them.**
> Many users drag windows to the edge of the screen, but using **Win + ←** and **Win + →** is much faster and more precise.

---

> 💡 **Your display isn't always frozen.**
> If your screen goes black after a graphics driver issue, pressing **Win + Ctrl + Shift + B** can restart the graphics driver without requiring a full system restart.
